# Secure-Hybrid-Enterprise-Network-Design-Engineering-Firm-
## Overview
This project presents a secure hybrid enterprise network architecture designed for a defense contractor handling Controlled Unclassified Information (CUI). The network follows CMMC 2.0 Level 2 and NIST SP 800-171 compliance requirements, focusing on security, reliability, and controlled access.
## Key Features
- Designed a hybrid network integrating on-premises infrastructure with cloud services
- Implemented Zero Trust architecture to enforce strict access control
- Network segmentation using VLANs to isolate sensitive systems
- Configured a secure DMZ for external-facing services
- Built high-availability firewall clusters (NGFW) for redundancy and protection
- Dual-ISP setup for failover and continuous network availability
## Security Implementation
- Encrypted site-to-site VPN connecting on-premises network to cloud environment
- Multi-Factor Authentication (MFA) enforced for secure remote access
- Deployed IDS/IPS for threat detection and prevention
- Bastion host implemented for secure administrative access
- Integrated logging and monitoring using SIEM (Azure Sentinel)
